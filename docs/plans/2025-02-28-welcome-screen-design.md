# YuGAgent 欢迎界面设计文档

**日期**: 2025-02-28
**设计者**: Claude
**状态**: 已批准

## 1. 概述

为 YuGAgent 的 TUI 界面设计一个赛博朋克风格的欢迎屏幕，在 `chat` 命令启动时显示，提升用户体验和品牌识别度。

## 2. 设计目标

1. **品牌识别度**: 创建独特的视觉标识，增强 YuGAgent 的品牌形象
2. **信息展示**: 清晰展示版本、模型、Session 等关键信息
3. **用户体验**: 提供友好的启动界面和操作提示
4. **技术一致性**: 使用现有的 Ink 技术栈，保持代码架构一致

## 4. 组件设计

### 4.1 组件结构

```
WelcomeMessage/
├── CatLogo.tsx         # 小猫 Logo + YuGAgent 文字
├── WelcomeInfo.tsx     # 版本、模型、Session 信息
└── WelcomeMessage.tsx  # 主组件（整合所有部分）
```

### 4.2 CatLogo 组件

**职责**: 渲染小猫 ASCII 艺术和 YuGAgent 文字

**特性**:
- 使用渐变色渲染猫耳朵轮廓
- "YU" 使用粉红色，"GAGENT" 使用青色
- 装饰线使用黄色

### 4.3 WelcomeInfo 组件

**职责**: 显示应用信息和操作提示

**特性**:
- 使用 emoji 图标增强视觉效果
- Session ID 只显示前 8 位 + 省略号
- 信息水平排列，节省垂直空间

### 4.4 WelcomeMessage 主组件

**职责**: 整合所有子组件，提供完整的欢迎界面

**Props**:
```typescript
interface WelcomeMessageProps {
  appName?: string;     // 应用名称，默认 'YuGAgent'
  version?: string;     // 版本号，默认 '2.0.0'
  model?: string;       // 模型名称
  sessionId?: string;   // Session ID
  maxWidth?: number;    // 最大宽度
}
```

## 5. 响应式设计

| 终端宽度 | 显示策略 |
|----------|----------|
| >= 80 字符 | 完整显示所有元素 |
| 60-80 字符 | 简化边框，保持核心内容 |
| < 60 字符 | 最小化版本，仅显示核心信息 |

## 6. 技术实现

### 6.1 技术栈

- **框架**: Ink (React for CLI)
- **语言**: TypeScript

## 7. 显示逻辑

欢迎界面在以下情况下显示：
- **显示条件**: `state.showWelcome === true` 且 `state.messages.length === 0`
- **隐藏条件**: 用户发送第一条消息后，设置 `showWelcome = false`
